Feedback/Bugs/Etc.

Please report bugs and feature suggestions via the issue tracker.

Feel free to fork the code and push changes to me. I'll accept them without much question (and your feature request will probably happen much more quickly).

Download

You can download this project in either zip or tar formats (although each package should be considered pre-release if their version does not yet exist on CTAN).

You can also clone the project with Git by running: 

$ git clone git://github.com/wspr/herries-press
